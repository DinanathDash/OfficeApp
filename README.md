# OfficeApp

Welcome to OfficeApp! This is an Android application designed to handle various office document formats, including PDF, Word, and Excel files.

## Features

- **PDF Handling**: View and interact with PDF documents.
- **Word processing**: Support for reading and processing Word documents (Doc/Docx).
- **Excel Support**: Handle spreadsheet data.
- **Secure**: Built with security best practices in mind.

## Tech Stack

- **Language**: Java
- **Platform**: Android
- **Key Libraries**:
    - **Apache POI**: For handling Microsoft Office documents.
    - **PDFBox**: For PDF processing.
    - **Android Jetpack**: For modern Android development.

## Setup & Installation

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dinanathdash/OfficeApp.git
    cd OfficeApp
    ```
2.  **Open in Android Studio:**
    - Launch Android Studio.
    - Select "Open an existing Android Studio project" and navigate to the `OfficeApp` directory.
3.  **Build the project:**
    - Wait for Gradle sync to complete.
    - Click the "Run" button (green play icon) to build and deploy to an emulator or device.
